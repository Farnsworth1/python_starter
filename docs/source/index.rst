Configuration of Project Environment
*************************************

Beamon is a database-based program for modeling static structures.
The simulation of such Structures are made with CALFEM-Python

Overview on How to Run Beamon
================================
1. Either install Anaconda or create a Python virtual environment to install the packages required
2. Install packages required using requirements.txt file
